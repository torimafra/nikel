<h1>NIKEL | Sistema de Controle Financeiro</h1>

Aplicação desenvolvida com HTML, CSS, JavaScript e Bootstrap, que permite o cadastro de usuários e o gerenciamento simples de entradas e gastos financeiros, com dados salvos diretamente no LocalStorage do navegador.
<br><br>Projeto final do curso <strong>CODAÍ 2.0 - FRONT END</strong> da Growdev.
<br>
<br>
<h2>Funcionalidades</h2>
<h3>Cadastro e Login</h3>
<ul>
  <li>Cadastro de novos usuários</li>
  <li>Validação de login utilizando dados armazenados no LocalStorage</li>
</ul>

<h3>Controle Financeiro</h3>
<ul>
  <li>Registro de entradas (receitas) e gastos (despesas)</li>
  <li>Armazenamento de todas as informações no LocalStorage</li>
  <li>Listagem automática das transações salvas</li>
  <li>Atualização dinâmica da interface a cada nova inserção</li>
  <li>Exibição do saldo com base nos valores cadastrados</li>
</ul>
